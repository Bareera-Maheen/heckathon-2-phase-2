# Quickstart Guide

## Prerequisites
- Python 3.13 or higher.

## Installation
No installation required. This is a standalone script.

## Running the Application
From the project root:
```bash
python3 src/main.py
```

## Usage
Once the application starts, you will see a prompt `>`. Enter commands as follows:

- **Add a task**: `add Buy groceries`
- **View tasks**: `list`
- **Complete a task**: `complete 1`
- **Remove a task**: `delete 1`
- **Exit**: `exit`

## Development
- **Run Tests**: `pytest`
- **Linting**: `pylint src` (or configured linter)
